Implementation and Resources

What technology will we use to create the app?

  • Flutter – Framework that provides the ability to create apps for both Android and IOS at the same time. Flutter will be used to create the user frontend of the app.
  • Firebase – Firebase is a cloud-based cloud database that can be used to store data and communicate with other services. It will be used for the projects backend, handling user authetification and realtime manipulation of data.
  • Visual Studio Code - A free, open-source, and highly customizable code editor developed by Microsoft, supporting a wide range of programming languages and development tools. Flutter is supported within VS Code.
  • Git/ GitHub - The projects version control will be handled by Git and GitHub.
